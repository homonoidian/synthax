    struct Sthx::Ctx

    • Sthx::Ctx
    • Struct
    • Value
    • Object

    Overview

    Represents the parsing context.

    • reader is the Char::Reader used for reading.
    • root is the current root of the parse tree (see Tree).

    Note that Ctx is a mutable struct if reader is interacted with, so beware of dragons.
